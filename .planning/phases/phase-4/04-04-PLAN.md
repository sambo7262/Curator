<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| slskd peer results → Candidate → gate | untrusted peer metadata is normalized to neutral Candidates before any decision |
| peer filename → staging path → purge/quarantine | a malicious filename must not escape the staging root (guarded by staging.assert_under_root from 04-02) |
| Readarr import fault → loop | a book outage must isolate to that book (ARR-02) |
| never-terminating download → slot exhaustion | a stalled peer must be cancelled, not held forever (D-01/ACQ-03) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-12 | DoS (resource exhaustion) | transfer watch loop | mitigate | No-progress stall detection (D-01) cancels (remove=true) after acq_stall_seconds of zero byte progress; the loop never holds a slot forever and falls to next/stuck |
| T-04-13 | Tampering / Elevation | staging purge/quarantine in the loop | mitigate | All purge/quarantine routes through staging.assert_under_root (04-02) anchored to settings.staging_root/quarantine_root; never a peer- or *arr-supplied path |
| T-04-14 | Tampering | importing unwanted/untrusted files | mitigate | The adapter pre-filters to the importable subset (empty-rejection resources) and execute_import lists exactly those files; core consumes that list without re-deriving it; the gate + adapter filter are the content-trust mitigations (no blind rescan) |
| T-04-15 | DoS (availability) | Readarr import fault in the loop | mitigate | Best-effort adapter returns safe defaults; the loop maps a book fault to a quarantine outcome for that item only — music path unaffected (ARR-02) |
| T-04-16 | Information Disclosure | secrets in acquire logs | mitigate | acquire.py logs only neutral GapItem identity + outcome strings; keys/tokens stay in the adapter/client headers, never logged |
| T-04-SC | Tampering | pip installs | accept | No new package (httpx/respx already pinned + approved). No install task. |
